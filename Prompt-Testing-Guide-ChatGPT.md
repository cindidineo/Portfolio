## PROMPT 1: Late Delivery

**Copy everything below and paste into ChatGPT:**

```
Role: You are a friendly, professional customer support agent for NeuroFive Store.
Context: A customer contacted us about late delivery. Their tone is frustrated. 
         Order/Issue ID: NF-8821. Product: Wireless Headphones.
Task: Write a support reply that resolves late delivery and makes the customer feel heard.
Format: 
1. Greeting + Empathy
2. Solution/Explanation 
3. Next Steps
4. Closing + Signature
Constraints: 
- Tone: Apologetic and helpful
- Length: 150 words max
- Do not promise refunds unless true = false
- Use the customer's name: Sarah
```

**Expected output should include:**
- Apology for the delay
- Explanation of what happened
- Clear delivery timeline
- Compensation (discount/voucher)
- Direct tracking link offered

## Testing Checklist

As you run each prompt, check:

- [ ] **Output matches expected content** — Does it include all the key elements listed above?
- [ ] **Tone is appropriate** — Does it match the constraint (apologetic, calm, enthusiastic, etc.)?
- [ ] **Word count is under limit** — ~150-200 words? (Paste output into Word counter)
- [ ] **Customer name used** — Is "Sarah" / "James" / etc. in the response?
- [ ] **Professional formatting** — Is it easy to copy/paste to a customer email?
- [ ] **Next steps are clear** — Can customer easily understand what to do next?

## Tips for Best Results

### Before Pasting:
- Make sure ChatGPT is in a fresh chat (not continuing from previous conversation)
- Use ChatGPT 3.5 or 4 (4 tends to be slightly better with tone matching)

### After Generating:
- **If the output is too long** → Add to the constraint: "- Length: 120 words max"
- **If the tone is off** → Rerun with more specific tone example: "Tone: Apologetic like saying 'I messed up and I'll fix it'"
- **If it's too robotic** → Add: "Make it sound like a real human wrote it, not a bot"

### Screenshot Tips:
- Use Ctrl+Shift+S (Windows) or Cmd+Shift+4 (Mac) to capture just the ChatGPT output
- Name files like: `Prompt1-LateDelivery-Output.png`
- Save to a folder called "Prompt-Testing-Screenshots" for your video

## Troubleshooting

**Q: ChatGPT output is too similar to all other prompts**
A: Add more specific details to the context. Instead of just "frustrated," say "Customer ordered 2 weeks ago, still no tracking info."

**Q: Output is too long**
A: In constraints, change word max to 120. Or add: "Make this concise and scannable"

**Q: Tone doesn't match**
A: Try rephrasing the tone constraint: Instead of "Apologetic" → "Sound like you made a mistake and you're taking full responsibility"

**Q: It's too formal**
A: Add to constraints: "Use casual language like you're talking to a friend"
